# Shell Basics

- [ ]（实际上是 test 命令）是 POSIX 标准的产物，而 [[]] 是 Bash/Zsh 的关键字（Keyword），后者在脚本编写中几乎全方位优于前者。

## file check

表达式,名称,核心逻辑,处理软链接（Symlink）的行为
[[-f $path]],普通文件判断,判断是否为 Regular File。,追踪（Follow）：如果链接指向一个文件，返回真；如果指向目录或断开，返回假。
[[-L $path]],符号链接判断,只判断路径是否为一个 Symlink。,不追踪：只要这个“壳子”是链接就返回真，无论它指向哪里（哪怕指向的是空地址）。
