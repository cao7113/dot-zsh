# dsh, also as dot-zsh

Personal customized zsh config.

## Usage

```
# switch profiles (zsh config)
dsh use [omz|ant|bare|play]

# dotrc links
dot ln /path/to/rc/file
ls -l ~/.zsh_dots/
```

## How it works?

- `task setup`
- new session auto-load `~/.zshrc` as soft-link
