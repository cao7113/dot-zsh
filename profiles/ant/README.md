# Ant short for antidote

a lightweight, fast, and modern Zsh plugin manager.

```
# plugin bundles home
antidote home

antidote path ohmyzsh/ohmyzsh

$ rm -rf $(antidote home)
$ rm ${ZDOTDIR:-~}/.zsh_plugins.zsh
```

## How it works

- translate plugins to bundles
- precompile to zsh_plugins.zsh
